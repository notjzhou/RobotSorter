# RobotSorter

### Prerequisites
This project needs Java, JDK 21+ is recommended in general, but there's so little in this project JDK 8+ can also work.
### If Java is not installed
Go to the [Oracle Java Downloads](https://www.oracle.com/java/technologies/downloads/) page.

### Building project
Run these from the repository root:
Linux
```./gradlew build```
Windows
```.\gradlew.bat build```

### Running tests *THIS IS WHAT TO RUN TO CHECK THAT MY CODE WORKS*
Run these from the repository root:
Linux
```./gradlew test```
Windows
```.\gradlew.bat test```

### *THIS IS WHERE TO LOOK AT MY CODE (sort method and the tests)*
- `src/main/java/org/example/RobotSorter.java` - sort method
- `src/test/java/com/example/RobotSorterTest.java` - unit tests
